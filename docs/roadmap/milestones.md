# Development Milestones

This document defines the **major development stages** of the project.

---

# Milestone 1 — Prototype Combat

Goal: validate core tactical gameplay.

Features:

* basic grid system
* character movement
* simple attack action
* combat scene prototype

---

# Milestone 2 — Combat Core

Goal: complete core tactical mechanics.

Features:

* turn order system
* ability targeting
* enemy AI behavior
* combat UI

---

# Milestone 3 — Gameplay Systems

Goal: introduce progression systems.

Features:

* character stats
* equipment
* inventory system
* basic quests

---

# Milestone 4 — Content Expansion

Goal: build playable game experience.

Features:

* multiple enemy types
* additional combat arenas
* story missions
* environmental interactions

---

# Milestone 5 — Polishing

Goal: improve player experience.

Features:

* visual feedback
* sound effects
* balancing
* performance optimization
