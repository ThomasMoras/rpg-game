# Prototypes

This document tracks gameplay prototypes used to validate design ideas.

---

# Prototype 1 — Grid System

Goal:

Validate grid movement and positioning.

Test elements:

* grid generation
* movement range
* pathfinding

Expected result:

Movement on grid feels responsive and clear.

---

# Prototype 2 — Combat Loop

Goal:

Validate turn-based combat flow.

Test elements:

* turn order
* player actions
* enemy actions

Expected result:

Combat loop feels smooth and understandable.

---

# Prototype 3 — Environmental Interaction

Goal:

Test interaction between abilities and terrain.

Examples:

* fire spreading from objects
* destructible environment elements

Expected result:

Environment becomes a meaningful tactical element.

---

# Prototype 4 — Enemy AI

Goal:

Evaluate tactical enemy decision making.

Test elements:

* movement decisions
* attack priorities
* ability usage

Expected result:

Enemies behave strategically without becoming predictable.
